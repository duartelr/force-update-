# force-update-
Este projeto tem como automatizar o processo de atualização do Windows e aplicativos reduzindo a necessidade de intervenções manuais  A solução utiliza:  PowerShell  PSWindowsUpdate  Winget  Task Scheduler (Agendador de Tarefas)  para executar atualizações periódicas de forma silenciosa, com geração de logs para auditoria e controle.
